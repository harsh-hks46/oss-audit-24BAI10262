## Project Description
This project is an audit of Python as an open-source software. It includes analysis of its origin, license, ecosystem, and comparison with proprietary software. It also includes 5 shell scripts demonstrating Linux concepts.

---

## Shell Scripts

### 1. System Identity Report
Displays system information like kernel, user, uptime, and OS details.

### 2. FOSS Package Inspector
Checks if Python is installed and displays version and details.

### 3. Disk and Permission Auditor
Shows directory permissions and disk usage.

### 4. Log File Analyzer
Analyzes log files and counts occurrences of a keyword.

### 5. Open Source Manifesto Generator
Generates a custom open-source philosophy statement.
